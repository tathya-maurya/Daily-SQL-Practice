# DAY-05: SQL Aggregate Functions Practice

This module contains practice queries based on **SQL Aggregate Functions** using the classic `EMP` table (Oracle SQL syntax).

## Topics Covered

- `COUNT()`
- `SUM()`
- `AVG()`
- `MAX()` / `MIN()`
- `DISTINCT`
- `LIKE` patterns
- Conditional filtering using `WHERE`

## Description

The queries focus on real-world employee data scenarios such as:

- Salary-based analysis
- Job-wise aggregation
- Department-wise employee counts
- Pattern matching on employee names
- Date-based filtering

## Purpose

- Strengthen understanding of aggregate functions
- Prepare for **SQL interviews & assessments**
- Improve query-writing accuracy and logic

## Notes

- Uses Oracle-style SQL
- Table used: `EMP`
