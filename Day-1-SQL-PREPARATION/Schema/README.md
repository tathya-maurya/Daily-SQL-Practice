# EMP Table Schema (Oracle & MySQL)

This folder contains the standard Oracle EMP table used for SQL practice.

## Files

## ORACLE
- `create_emp_table_oracle.sql` → Creates EMP table
- `insert_emp_data_oracle.sql` → Inserts sample employee data

## Usage

1. Run `create_emp_table_oracle.sql`
2. Run `insert_emp_data_oracle.sql`
3. Execute day-wise SQL queries using EMP table

## MYSQL

- `create_emp_table_mysql.sql` → Creates EMP table in MySQL
- `insert_emp_data_mysql.sql` → Inserts sample employee data
